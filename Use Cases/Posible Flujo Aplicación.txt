Flujo de la aplicación.

1. GESTION DE PRODUCTOS -Clickamos en productos y aparece el grid view del mockup con todos los productos, se puede buscar por SKU (leer el caso de uso).

-> Listar

-Desde este grid se puede añadir, editar y borrar un producto (caso de uso a parte, con un nuevo boceto pop-up)).
-> Crear
-> Consultar
-> Modificar/Editar
-> Eliminar
-> Categoría, label, atributo de usuario, de sistema, específico se añade aquí.

Los atributos de usuario se gestionan (añaden hasta 5, con su tipo y nombre) en el breadcrumb de Atributos.
2. GESTIÓN DE ATRIBUTOS DE USUARIO (MAX 5)
-> CRUD (TIPO DEL ATRIBUTO, NOMBRE)
	-> SI EL TIPO ES STRING, CARACTER MAX 250
